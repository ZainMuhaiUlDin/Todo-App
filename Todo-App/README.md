-> Todo App

A persistent, full-featured Todo web app built with HTML, CSS, and JavaScript.  
Tasks are saved in your browser's localStorage — they stay even after you close the tab.

---

## ▶️ How to Run (Step by Step)

### Option 1 — Open Directly in Browser (Easiest)

1. Download or unzip the project folder (e.g. `todo-app/`)
2. Open the folder on your computer
3. Find the file named `index.html`
4. Double-click `index.html` — it opens in your default browser
5. That's it! The app is running ✅

> Works in Chrome, Firefox, Edge, Brave — any modern browser.

---

### Option 2 — Open with VS Code Live Server (Recommended for development)

1. Open VS Code
2. Go to `File → Open Folder` and select the `todo-app` folder
3. Install the Live Server extension (search in Extensions tab)
4. Right-click `index.html` in the VS Code file explorer
5. Click "Open with Live Server"
6. The app opens at `http://127.0.0.1:5500`

## 🔧 No Installation Required

This is a **pure frontend app** — no Node.js, no npm, no backend needed.  
Just open `index.html` and it works.

---

- You can change any text in `index.html` (headings, placeholder text, etc.)
- You can change colors in `style.css` by editing the `:root` variables at the top
- Tasks survive page refresh and browser close because of `localStorage`
